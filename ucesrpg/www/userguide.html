User Guide

Welcome! This app allows you to answer quiz questions about your surroundings! You can access a question when you are within 25m of one.

In order to access questions, first select the blue explore button.

Wait for the app to find your location before proceeding.

The nearby selection function will not work unless your location has been found

Next select the green track_changes button in order to find which questions are nearby!

Questions that are within range will appear in green, while questions that are too far away will appear in red.

The map style can be changed from within the map area

This app will only function if the httpServer.js is running. This server can be found at https://github.com/rpGIS/server
